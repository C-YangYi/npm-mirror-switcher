# NPM Mirror Switcher

A practical VS Code extension to quickly switch npm / pnpm / yarn registry.

## Features

- Status bar display current registry
- One-click switch registry
- Auto detect project package manager

## Usage

- Click status bar: switch registry
- Or use Command Palette:
  `切换包管理器镜像（npm / pnpm / yarn）`

## Supported Registries

- Aliyun
- NPM
